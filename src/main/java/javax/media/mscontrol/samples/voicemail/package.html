This package contains a simple voicemail application.

The starting point is VoiceMailServlet.java , which is the SIP Servlet interfacing.
Based on the caller's identity, one version of the dialog is selected:

  • CoreMailSession.java is a simple user-interaction dialog, using DTMF patterns and messages defined by core objects.
    The callers are required to enter a PIN code, and can leave messages, or read messages that were deposited previously.
    CoreMailbox.java and CoreMailboxMgr.java are ancillary classes, not JSR309-related.

  • VxmlMailSession.java shows how to launch a vxml interpreter to handle the user-interaction. The vxml script is
    fetched on an http server; the script itself is not included, and so the service is not further described.

The remaing file, VoiceMailSession.java, is a common part to the two previous ones. It contains the management
of the NetworkConnection, which is same in both cases.
